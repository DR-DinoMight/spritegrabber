> A simple Sprite Grabber and animator

## Install

```sh
npm install
```

## Usage

```sh
npm run grab
```
this will grab you 1000 sprite images you may want to tweak as needed. `line 19` or `index.js`

```sh
npm run web
```
this is the simple animation around the screen. To edit go to the `web.js`
